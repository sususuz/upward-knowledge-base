## 响应结构

### 成功响应 (200)

```json
{
  "code": 0,
  "message": "success",
  "data": {
    "list": [],
    "total": 100
  }
}
```

### 异常响应

| 错误码 | HTTP状态码 | 含义 | 前端处理方式 |
|-------|-----------|------|------------|
| 1001 | 400 | 参数校验失败 | 提示具体字段 |
| 1002 | 401 | 未登录 | 跳转登录页 |
| 1003 | 403 | 无权限 | 提示"无操作权限" |
| 1004 | 429 | 请求过于频繁 | 提示"操作太频繁" |
| 5000 | 500 | 服务器错误 | 提示"系统异常" |
