# 测试用例覆盖缺口分析

## 原用例覆盖不足点

### 群管理模块（24-28页）
- 缺少**群管理入口角色权限**：普通用户、管理员、群主进入群管理界面后的元素差异
- 缺少**成员头像点击进个人主页**（标注36）
- 缺少**查看更多**进入完整成员列表
- 缺少**群昵称页面**：仅自己可见的本地昵称、输入框、完成按钮、为空/超长校验
- 缺少**管理权转让**：搜索群成员、单选限制、确认转让、转让后权限变化
- 缺少**群主不可退出**的拦截场景（弹窗提示"请先转让群主权限"）
- 缺少**管理员退出后自动撤销管理员身份**场景
- 缺少**关闭/重启/解散**的二次确认与状态流转
- 缺少**解散后3天自动消失**、卡片标灰规则
- 缺少**加入审核开关**的即时生效
- 缺少**自动关闭/归档**的默认值与规则

### 成员管理模块（13,33-36页）
- 邀请成员：缺少**搜索框**、**已是成员不可选**、**可选人数上限=群剩余容量**、**群满员拦截**、**接受邀请流程**
- 移除成员：缺少**批量移除入口**、**不可移除群主**、**移除管理员后撤销身份**、**系统消息与推送**
- 申请列表：缺少**按时间排序**、**全部同意/全部拒绝的二次确认**、**红点提示**、**满员时同意失败**、**申请理由超长截断**
- 用户申请加入：缺少**理由输入框字数统计**、**提交后Toast**、**无需审核直接跳过此页**

### 群公告模块（29-32页）
- 缺少**空状态展示**（"暂无群公告"）
- 缺少**公告卡片要素**：标题、摘要、时间、附件标识、置顶标识
- 缺少**操作菜单**：取消置顶、设为新人必看、删除公告
- 缺少**置顶替换规则**：最多1条置顶，新置顶替换旧置顶
- 缺少**新人必看默认设置**
- 缺少**公告数量上限5个**
- 缺少**新建页面字段**：标题必填、内容必填、1万字限制、附件插入类型
- 缺少**文件类型边界**：图片jpg/png/webp、视频mp4/mov、文档pdf/doc/docx/xls/xlsx/ppt/pptx/txt、运动数据gpx/tcx/fit（二期）
- 缺少**附件大小边界**：图片≤20MB、视频≤200MB、文档≤100MB、总量≤300MB
- 缺少**本期不支持@人/定位/轨迹**
- 缺少**公告内容页**：编辑、删除、附件预览/下载
- 缺少**群关闭后公告只读**、**群解散后公告删除**

## 改进目标
- 用例总数从 43 条提升到 **100+ 条**
- P0主流程冒烟单独 Sheet，18-20条
- 其他用例覆盖：功能测试、交互测试、UI测试、异常测试、兼容测试
- 每个页面关键路径和规则边界都要有对应用例
