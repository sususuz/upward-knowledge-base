## 8. 我的圈子入口

### 页面概述
我的圈子模块主入口，聚合已加入的圈子、管理的圈子、发现圈子三个核心入口。

### 组件清单

| 组件 | 说明 |
|------|------|
| 加入的圈子 | 入口，显示已加入圈子数量 |
| 管理的圈子 | 入口，显示管理的圈子数量 |
| 发现圈子 | 入口，进入圈子推荐/搜索 |
| 搜索图标 | 跳转搜索页 |
| 创建圈子入口 | 跳转创建圈子页 |

### 用户操作路径

| 操作 | 结果 |
|------|------|
| 点击加入的圈子 | 进入已加入圈子列表 |
| 点击管理的圈子 | 进入管理圈子列表 |
| 点击发现 | 进入圈子发现页 |
| 点击搜索 | 进入搜索页 |
| 点击创建 | 进入创建圈子页 |

## 9. 我的圈子-加入的

### 页面概述
展示用户已加入的所有圈子列表，按最新消息时间倒序排列。

### 组件清单

| 组件 | 说明 |
|------|------|
| 圈子列表 | 圈子卡片列表 |
| 未读红点 | 未读消息提示 |
| 消息数角标 | 最多显示 99+ |
| 空状态 | 无加入圈子时的引导 |

### 数据字段

| 字段 | 类型 | 说明 |
|------|------|------|
| 圈子名称 | 文本 | 圈子名称 |
| 最新消息摘要 | 文本 | 最后一条消息摘要 |
| 最新消息时间 | 时间 | 用于倒序排序 |
| 未读数量 | 数字 | 未读消息数 |
| 圈子头像 | 图片 | 圈子封面/头像 |

### 业务规则
- 按最新消息时间倒序排序
- 未读消息显示红点，点击进入后消失
- 群聊条数实时显示，最多 99+

## 10. 我的圈子-管理的

### 页面概述
展示用户作为群主或管理员管理的圈子列表。

### 组件清单

| 组件 | 说明 |
|------|------|
| 管理圈子列表 | 展示管理的圈子 |
| 管理角色标识 | 群主/管理员 |
| 空状态 | 无管理圈子时的引导 |

### 业务规则
- 普通用户创建圈子后自动成为群主
- 被设为管理员的用户也会出现在管理列表中

## 12. 我的圈子-发现

### 页面概述
圈子推荐发现页，展示同城、同标签、热门圈子。

### 组件清单

| 组件 | 说明 |
|------|------|
| 搜索框 | 搜索圈子 |
| 推荐列表 | 同城/标签/热门圈子 |
| 圈子卡片 | 名称、成员数、简介、封面、圈主 |
| 加入按钮 | 快速加入/申请加入 |

### 推荐排序逻辑
1. 优先展示同城圈子
2. 按用户标签匹配度推荐
3. 按圈子热度和创建时间推荐

### 业务规则
- 用户群可一键加入或申请加入
- 赛事/活动群默认点击加入需审核 ⚠️待确认
- 已加入的圈子显示"已加入"或隐藏

## 42. 我的圈子-搜索

### 页面概述
圈子搜索页，支持按圈子名称和圈主名称模糊搜索。

### 组件清单

| 组件 | 说明 |
|------|------|
| 搜索框 | 输入关键字 |
| Tab 页签 | 圈子/圈主/全部 |
| 搜索结果列表 | 展示匹配的圈子 |
| 高亮关键字 | 匹配文字高亮显示 |
| 清空按钮 | 清空搜索关键字 |

### 数据字段

| 字段 | 类型 | 说明 | 校验规则 |
|------|------|------|---------|
| 搜索关键字 | 文本 | 圈子名称/圈主名称 | 实时搜索 |
| 搜索范围 | 枚举 | 圈子/圈主/全部 | Tab 切换保留关键字 |

### 业务规则
- 对圈子名称和圈主名称模糊搜索
- 输入文字后实时搜索
- 搜索结果中高亮关键字
- 切换 Tab 页签后保持关键字
