## 16. 群聊界面

### 页面概述
群聊主界面，展示消息列表、输入区、置顶消息区、群名称和返回按钮。支持发送多种消息类型。

### 截图区
- 默认态：正常消息列表、输入框、功能按钮
- 加载中：消息历史加载中
- 空数据：新群无消息时显示引导
- 错误态：网络异常时显示重试

### 组件清单

| 组件名称 | 类型 | 说明 |
|---------|------|------|
| 返回按钮 | 导航 | 返回上一级页面 |
| 群名称 | 文本 | 显示当前群聊名称 |
| 更多按钮 | 按钮 | 进入群管理界面 |
| 置顶消息区 | 卡片 | 展示置顶消息/公告，可收起 |
| 消息列表 | 列表 | 展示文本/图片/视频/语音/文件/链接/位置/卡片消息 |
| 输入框 | 输入 | 文本输入区，支持表情、快捷发送 |
| 语音按钮 | 按钮 | 切换语音输入模式 |
| 加号按钮 | 按钮 | 打开图片/文件/位置等更多发送选项 |

### 用户操作路径

| 操作 | 触发方式 | 结果 |
|------|---------|------|
| 发送文本消息 | 输入内容后点击发送 | 消息出现在消息列表 |
| 发送语音 | 长按语音按钮 | 录制并发送语音消息 |
| 发送图片 | 点击加号 → 选择图片 | 发送图片消息 |
| 发送文件 | 点击加号 → 选择文件 | 发送文件消息 |
| 查看消息详情 | 点击消息内容 | 放大图片、播放视频/语音等 |
| 长按消息 | 长按任意消息 | 弹出消息操作菜单 |
| 查看置顶 | 点击置顶消息区 | 跳转或展开置顶内容 |
| 进入群管理 | 点击右上角更多 | 跳转群管理界面 |

### 数据字段

| 字段 | 类型 | 说明 | 校验规则 |
|------|------|------|---------|
| 消息内容 | 文本 | 文本消息内容 | 非空，≤2000字 |
| 图片 | 文件 | 图片消息 | 单张 ≤10MB，最多9张 |
| 视频 | 文件 | 视频消息 | ≤500MB，≤5分钟 |
| 语音 | 文件 | 语音消息 | ≤60秒 |
| 文件 | 文件 | 文件消息 | ≤200MB |
| 位置 | 对象 | 位置信息 | 含地名、详细地址、经纬度 |

### 关联页面

- 群聊-管理员菜单（18）
- 群管理界面（24）

## 17. 群聊-菜单

### 页面概述
普通用户长按消息后弹出的操作菜单，包含引用、复制、多选、撤回、转发、举报等操作。

### 用户操作路径

| 操作 | 权限 | 说明 |
|------|------|------|
| 引用 | 全员 | 引用当前消息 |
| 复制 | 全员 | 复制文本/链接/位置 |
| 多选 | 全员 | 进入多选模式 |
| 撤回 | 自己2分钟内 | 撤回自己的消息 |
| 转发 | 全员 | 转发消息到其他会话 |
| 举报 | 他人消息 | 举报违规消息 |
| 保存 | 全员 | 保存图片/视频/文件/语音 |

### 数据字段

| 字段 | 说明 |
|------|------|
| 操作类型 | 引用/复制/多选/撤回/转发/举报/保存 |
| 消息ID | 被操作消息的唯一标识 |
| 发送者ID | 用于判断操作权限 |

## 18. 群聊-管理员菜单

### 页面概述
群主/管理员长按消息后显示的操作菜单，在普通菜单基础上额外支持置顶、删除、撤回他人消息。

### 用户操作路径

| 操作 | 权限 | 说明 |
|------|------|------|
| 置顶 | 群主/管理员 | 置顶消息到群顶部 |
| 删除 | 群主/管理员 | 删除他人消息 |
| 撤回他人消息 | 群主/管理员 | 无时间限制撤回 |
| 其他操作 | 同普通菜单 | 引用/复制/多选/转发/举报 |

## 19. 群聊-消息置顶

### 页面概述
展示置顶消息的区域，最多 5 条置顶消息 + 1 条置顶公告。新成员入群会收到置顶消息全文推送。

### 组件清单

| 组件 | 说明 |
|------|------|
| 置顶消息列表 | 展示最多 5 条消息摘要 |
| 展开/收起按钮 | 收起后只显示一条摘要 |
| 取消置顶入口 | 群主/管理员可取消 |

### 数据字段

| 字段 | 说明 |
|------|------|
| 置顶消息ID | 唯一标识 |
| 置顶时间 | 用于替换最早置顶 |
| 置顶者ID | 操作人 |

---

## 20. 群聊-公告置顶

### 页面概述
置顶公告在群聊顶部展示，与普通置顶消息区分开，最多 1 条。

### 组件清单

| 组件 | 说明 |
|------|------|
| 公告标题 | 置顶公告标题 |
| 公告摘要 | 部分内容摘要 |
| 查看详情 | 点击进入公告内容页 |
| 取消置顶 | 群主/管理员可操作 |

## 21. 群聊-置顶收起

### 页面概述
置顶消息收起后的状态，仅展示一条置顶消息摘要，点击可展开查看全部。

## 22. 群聊-多选

### 页面概述
消息多选模式，支持批量转发、删除、举报。

### 用户操作路径

| 操作 | 说明 | 限制 |
|------|------|------|
| 选择消息 | 点击消息左侧复选框 | 已撤回、系统消息不可选 |
| 逐条转发 | 批量逐条转发到目标会话 | 最多选 99 条 |
| 合并转发 | 合并为一条转发消息 | 最多选 99 条 |
| 删除 | 批量删除 | 仅群主/管理员 |
| 举报 | 批量举报 | 不可举报自己的消息 |

## 23. 群聊-举报

### 页面概述
举报消息页面，选择举报理由并补充说明。

### 数据字段

| 字段 | 类型 | 说明 | 校验规则 |
|------|------|------|---------|
| 举报理由 | 单选 | 必选 | 10 个预设选项 + 其他 |
| 其他说明 | 文本 | 选填 | ≤200字 |
| 补充说明 | 文本 | 选填 | ≤500字 |
| 截图证据 | 图片 | 选填 | 最多3张 |

### 业务规则
- 三级处理流程：L1 群主审核 → L2 平台升级 → L3 平台反馈
- 选择诈骗/政治敏感/色情低俗直接跳过 L1
- 反恶意举报：1h 最多 10 条，同一消息只能举报 1 次

## 38. 群聊-按住说话

### 页面概述
语音输入的按住状态，显示录音动画和"松开结束"提示。

### 数据字段

| 字段 | 说明 |
|------|------|
| 语音时长 | 实时显示，≤60秒 |
| 录音状态 | 按住录音、松开发送、上滑取消 |

---

## 39. 群聊-语音发送

### 页面概述
语音消息发送后的展示状态，显示语音时长和播放按钮。

### 数据字段

| 字段 | 说明 |
|------|------|
| 语音时长 | 显示秒数，如 50"、1'33" |
| 播放状态 | 未播放/播放中/已播放 |
| 已读标记 | 对方是否已播放 |

---

## 41. 群聊-转发

### 页面概述
选择转发目标会话的页面，支持选择多个群/好友。

### 用户操作路径

| 操作 | 说明 | 限制 |
|------|------|------|
| 选择目标 | 选择要转发的群/人 | 最多 9 个 |
| 添加附言 | 输入转发附言 | ≤200字 |
| 确认转发 | 发送转发消息 | 逐条转发 |

### 数据字段

| 字段 | 类型 | 说明 |
|------|------|------|
| 目标会话 | 数组 | 群/好友ID列表 |
| 附言 | 文本 | ≤200字 |
