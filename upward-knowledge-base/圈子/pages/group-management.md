## 24. 群管理界面

### 页面概述
群管理主页面，展示群基本信息、成员入口、群昵称、申请审批、退出群聊等。普通用户和管理员/群主看到的入口不同。

### 截图区
- 默认态：普通用户视角，可见成员、群昵称、退出群聊
- 管理员视角：额外显示申请审批、添加成员等
- 群主视角：额外显示群主管理入口

### 组件清单

| 组件名称 | 类型 | 说明 | 可见角色 |
|---------|------|------|----------|
| 返回按钮 | 导航 | 返回群聊界面 | 全员 |
| 页面标题 | 文本 | 群管理 | 全员 |
| 成员列表区 | 列表 | 展示前 N 个成员头像 | 全员 |
| + 按钮 | 按钮 | 添加/邀请成员 | 管理员、群主 |
| 查看更多 | 按钮 | 进入完整成员列表 | 全员 |
| 申请加入审批 | 入口 | 入群申请列表 | 管理员、群主 |
| 群昵称 | 入口 | 修改群昵称 | 全员 |
| 退出群聊 | 按钮 | 退出当前群 | 普通用户、管理员 |
| 群主管理入口 | 入口 | 群主专属管理页 | 群主 |

### 用户操作路径

| 操作 | 权限 | 结果 |
|------|------|------|
| 点击成员头像 | 全员 | 进入该成员个人主页 |
| 点击 + | 管理员、群主 | 进入邀请成员页 |
| 点击申请加入审批 | 管理员、群主 | 进入申请列表 |
| 点击群昵称 | 全员 | 进入群昵称设置页 |
| 点击退出群聊 | 普通用户、管理员 | 弹出确认，退出群聊 |
| 点击群主管理入口 | 群主 | 进入群管理-群主页 |

### 数据字段

| 字段 | 类型 | 说明 |
|------|------|------|
| 成员头像 | 图片 | 成员头像列表 |
| 成员昵称 | 文本 | 成员昵称 |
| 角色标识 | 文本 | 群主/管理员/普通成员 |
| 申请数量 | 数字 | 待审批申请数，显示红点 |

### 关联页面
- 群成员管理-邀请成员（33）
- 群成员管理-申请列表（36）
- 群管理-群主（25）
- 群昵称（28）

## 25. 群管理-群主

### 页面概述
群主专属管理页面，包含加入审核开关、自动关闭设置、管理权转让、关闭/重启/解散群聊。

### 组件清单

| 组件名称 | 类型 | 说明 |
|---------|------|------|
| 加入群需要审核 | 开关 | 控制入群是否需要审核 |
| 活动/榜单结束后14天自动 | 选项 | 活动/榜单群默认自动关闭 |
| 赛事结束后30天自动 | 选项 | 赛事群默认自动归档 |
| 群主管理权转让 | 入口 | 转让群主身份 |
| 群管理员 | 入口 | 设置管理员 |
| 关闭群聊 | 按钮 | 关闭群聊，保留历史消息 |
| 重启群聊 | 按钮 | 重启已关闭的群聊 |
| 解散群聊 | 按钮 | 永久解散群聊 |

### 用户操作路径

| 操作 | 结果 | 确认方式 |
|------|------|----------|
| 切换加入审核开关 | 改变入群审核状态 | 即时生效 |
| 点击群主管理权转让 | 进入转让页面 | 选择新群主后二次确认 |
| 点击关闭群聊 | 群变为关闭状态 | 二次确认 |
| 点击重启群聊 | 恢复运营 | 二次确认 |
| 点击解散群聊 | 群永久解散 | 二次确认 |

### 数据字段

| 字段 | 类型 | 说明 | 校验规则 |
|------|------|------|---------|
| 加入审核开关 | 布尔 | 是否开启入群审核 | 默认赛事/活动群开启 |
| 自动关闭设置 | 枚举 | 14天/30天/不自动 | 按圈子类型默认 |
| 新群主ID | 字符串 | 转让对象 | 必须是群内成员 |

### 业务规则
- 加入群需要审核：赛事/活动群默认点击加入需审核 ⚠️待确认；报名加入直接通过
- 活动/榜单结束后 14 天自动关闭群；赛事结束后 30 天自动归档
- 群主不可直接退群，需先转让
- 解散群聊后全部消息、文件永久删除

## 26. 群管理员设置

### 页面概述
群主设置管理员页面，展示群内普通成员列表，可选择最多 3 个管理员。

### 组件清单

| 组件 | 说明 |
|------|------|
| 成员列表 | 展示普通成员（不包含群主） |
| 复选框 | 选择/取消管理员 |
| 已选数量 | 显示当前已选管理员数量 |
| 保存按钮 | 保存管理员设置 |

### 数据字段

| 字段 | 类型 | 说明 | 校验规则 |
|------|------|------|---------|
| 管理员ID列表 | 数组 | 最多3个 | 超出上限提示 |
| 现有管理员 | 数组 | 默认选中 | 不可取消需二次确认 |

### 业务规则
- 最多 3 个管理员
- 不显示群主本人
- 现存管理员默认被选择
- 超出上限提示"管理员人数超过上限"

## 27. 管理权转让

### 页面概述
群主选择新群主页面，每次只能选择一个转让对象。

### 数据字段

| 字段 | 类型 | 说明 | 校验规则 |
|------|------|------|---------|
| 新群主ID | 字符串 | 转让对象 | 必须是群内成员，每次只能选1个 |

### 业务规则
- 转让后原群主变为普通成员
- 新群主获得全部群主权限
- 触发系统消息："群主已由 张三 转让给 李四"

## 28. 群昵称

### 页面概述
设置当前群在本地的显示昵称，仅影响自己看到的群名称。

### 数据字段

| 字段 | 类型 | 说明 | 校验规则 |
|------|------|------|---------|
| 群昵称 | 文本 | 本地显示名称 | 建议 ≤20字符 |

### 业务规则
- 普通用户、管理员、群主均可修改
- 仅影响当前用户本地显示，不影响群真实名称
