# 页面知识：动态发布模块

> 模块：动态发布
> 包含页面：动态发布-添加圈子（1）、发布的动态-圈子入口（2）、动态发布-选择圈子（37）
> 关联业务规则：`../business-rules/circle-types.md`

---

## 1. 动态发布-添加圈子

### 页面概述
发布动态时选择关联的圈子，将动态与圈子关联。

### 组件清单

| 组件 | 说明 |
|------|------|
| 已加入圈子列表 | 展示用户已加入的圈子 |
| 选择框 | 选择要关联的圈子 |
| 确认按钮 | 确认关联 |

### 数据字段

| 字段 | 类型 | 说明 | 校验规则 |
|------|------|------|---------|
| 关联圈子ID | 字符串 | 选择的圈子 | 暂时只支持添加一个圈子 |

### 业务规则
- 动态发布暂时只支持添加一个圈子
- 只能选择已加入的圈子

---

## 2. 发布的动态-圈子入口

### 页面概述
已发布的动态中展示关联的圈子入口，点击可进入圈子/群聊。

### 组件清单

| 组件 | 说明 |
|------|------|
| 圈子名称 | 显示关联圈子名称 |
| 圈子入口 | 点击进入圈子 |

### 用户操作路径

| 操作 | 结果 |
|------|------|
| 点击圈子入口 | 进入圈子详情/群聊 |

---

## 37. 动态发布-选择圈子

### 页面概述
选择圈子弹窗或独立页面，从已加入圈子中选择关联。

### 组件清单

| 组件 | 说明 |
|------|------|
| 搜索框 | 搜索已加入圈子 |
| 圈子列表 | 展示已加入圈子 |
| 确认选择 | 确认关联 |

### 数据字段

| 字段 | 类型 | 说明 | 校验规则 |
|------|------|------|---------|
| 圈子ID | 字符串 | 选择的圈子 | 只能选一个 |
