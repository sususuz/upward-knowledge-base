# 页面知识：群公告模块

> 模块：群公告
> 包含页面：群公告-展示页-空白（29）、群公告-展示页-非空白（30）、群公告-新建页面（31）、群公告-内容页（32）
> 关联业务规则：`../business-rules/announcements.md`、`../business-rules/permissions.md`

---

## 29. 群公告-展示页-空白

### 页面概述
当群公告数量为 0 时展示，引导群主/管理员创建公告。

### 组件清单

| 组件 | 说明 | 可见角色 |
|------|------|----------|
| 空状态提示 | "暂无群公告" | 全员 |
| 新建群公告按钮 | 创建新公告 | 群主、管理员 |

### 用户操作路径

| 操作 | 权限 | 结果 |
|------|------|------|
| 点击新建群公告 | 群主、管理员 | 进入新建页面 |

## 30. 群公告-展示页-非空白

### 页面概述
展示已有公告列表，支持查看详情、新建公告、公告操作菜单。

### 组件清单

| 组件 | 说明 | 可见角色 |
|------|------|----------|
| 公告卡片 | 标题、摘要、时间、附件标识 | 全员 |
| 置顶标识 | 置顶公告特殊标识 | 全员 |
| 更多菜单 | 取消置顶/设为新人必看/删除 | 群主、管理员 |
| 新建群公告按钮 | 创建新公告 | 群主、管理员 |

### 用户操作路径

| 操作 | 权限 | 结果 |
|------|------|------|
| 点击公告卡片 | 全员 | 进入公告内容页 |
| 点击更多菜单 | 群主、管理员 | 弹出操作选项 |
| 点击新建群公告 | 群主、管理员 | 进入新建页面 |
| 取消置顶 | 群主、管理员 | 取消该公告置顶 |
| 设为新人必看 | 群主、管理员 | 指定新人入群必看 |
| 删除公告 | 群主、管理员 | 二次确认后删除 |

### 数据字段

| 字段 | 类型 | 说明 |
|------|------|------|
| 公告ID | 字符串 | 唯一标识 |
| 公告标题 | 文本 | 公告标题 |
| 公告摘要 | 文本 | 前 N 字摘要 |
| 发布时间 | 时间 | 发布/更新时间 |
| 是否置顶 | 布尔 | 是否置顶 |
| 附件数量 | 数字 | 图片/文件/视频数量 |

## 31. 群公告-新建页面

### 页面概述
创建或编辑公告，支持文本、图片、文件、链接输入。

### 组件清单

| 组件 | 说明 |
|------|------|
| 标题输入框 | 公告标题 |
| 内容编辑器 | 公告正文，最多1万字 |
| 图片插入 | 插入图片 |
| 文件插入 | 插入文件 |
| 链接插入 | 插入链接 |
| 发布按钮 | 保存并发布公告 |
| 置顶开关 | 是否同时置顶 |

### 数据字段

| 字段 | 类型 | 说明 | 校验规则 |
|------|------|------|---------|
| 公告标题 | 文本 | 必填 | 非空 |
| 公告内容 | 文本 | 必填 | ≤10000字 |
| 图片 | 文件数组 | 选填 | 单张≤20MB，最多10张 |
| 文件 | 文件数组 | 选填 | 单个≤100MB，最多2个 |
| 视频 | 文件 | 选填 | ≤200MB，1个 |
| 链接 | 文本 | 选填 | 有效URL |
| 是否置顶 | 布尔 | 选填 | 置顶会替换旧置顶 |
| 新人必看 | 布尔 | 选填 | 默认 true |

### 业务规则
- 公告数量最多 5 个，超出后不可新建
- 单公告附件总大小 ≤300MB
- 本期不支持 @ 人、轨迹、定位

## 32. 群公告-内容页

### 页面概述
展示公告完整内容，包括标题、正文、附件、时间等。

### 组件清单

| 组件 | 说明 |
|------|------|
| 标题 | 公告标题 |
| 发布时间 | 发布者+时间 |
| 正文内容 | 完整文本、图片、文件、链接 |
| 附件列表 | 可下载/预览的附件 |
| 编辑按钮 | 群主、管理员可见 |
| 删除按钮 | 群主、管理员可见 |

### 用户操作路径

| 操作 | 权限 | 结果 |
|------|------|------|
| 点击附件 | 全员 | 下载或预览 |
| 点击编辑 | 群主、管理员 | 进入编辑模式 |
| 点击删除 | 群主、管理员 | 二次确认后删除 |
