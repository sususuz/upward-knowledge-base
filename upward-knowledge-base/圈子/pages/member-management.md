## 13. 用户申请加入

### 页面概述
用户在圈子发现页或用户主页点击"立即加入"后，填写申请理由提交入群申请的页面。

### 截图区
- 默认态：申请理由输入框，提交按钮
- 提交成功：Toast 提示"已经成功提交申请"
- 群满员态：提示"圈子人数已达上限"

### 组件清单

| 组件 | 类型 | 说明 |
|------|------|------|
| 圈子信息 | 展示 | 展示目标圈子名称、简介 |
| 申请理由输入框 | 多行输入 | 填写入群理由，≤1000字，不支持图片 |
| 字数统计 | 文本 | 实时显示已输入字数 |
| 提交按钮 | 按钮 | 提交申请 |

### 用户操作路径

| 操作 | 触发方式 | 结果 |
|------|---------|------|
| 输入申请理由 | 文本输入 | 实时字数统计 |
| 点击提交 | 点击按钮 | Toast"已经成功提交申请"，返回上一页 |
| 取消 | 返回 | 不提交，返回上一页 |

### 数据字段

| 字段 | 类型 | 说明 | 校验规则 |
|------|------|------|---------|
| 申请理由 | 文本 | 入群理由 | ≤1000字，不支持图片 |
| 目标圈子ID | 字符串 | 要加入的圈子 | — |

### 业务规则
- 申请理由不超过 1000 字，不支持图片
- 提交后 Toast 提示"已经成功提交申请"
- 群主/管理员收到申请通知（红点+站内通知）
- 无需审核的圈子（如报名加入）直接加入，跳过此页面

### 关联页面
- 我的圈子-发现（12）
- 群成员管理-申请列表（36）

## 33. 群成员管理-邀请成员

### 页面概述
邀请新成员加入群聊，从通讯录或好友列表中选择要邀请的人。

### 组件清单

| 组件 | 说明 |
|------|------|
| 搜索框 | 搜索好友/联系人 |
| 联系人列表 | 展示可邀请的联系人 |
| 已选区 | 展示已选联系人 |
| 确认邀请按钮 | 发送邀请 |

### 数据字段

| 字段 | 类型 | 说明 | 校验规则 |
|------|------|------|---------|
| 邀请对象ID列表 | 数组 | 被邀请人ID | 不可包含已加入成员；不超过剩余名额 |
| 邀请附言 | 文本 | 可选 | 默认无 |

### 业务规则
- 可选人数上限为群当前剩余可容纳人数
- 已是群成员的用户不可被选择
- 邀请发出后被邀请人收到站内通知

## 34. 群成员管理-移除成员

### 页面概述
移除单个成员，仅管理员/群主可操作，需二次确认。

### 组件清单

| 组件 | 说明 |
|------|------|
| 成员列表 | 展示可移除的成员 |
| 移除按钮 | 移除指定成员 |
| 批量移除入口 | 跳转批量移除模式 |

### 用户操作路径

| 操作 | 结果 | 确认方式 |
|------|------|----------|
| 点击移除 | 弹出二次确认 | 确认后移除 |
| 点击批量移除 | 进入批量移除页 | — |

### 数据字段

| 字段 | 类型 | 说明 |
|------|------|------|
| 被移除成员ID | 字符串 | 要移除的成员 |
| 移除原因 | 文本 | 可选，用于系统记录 |

### 业务规则
- 不可移除群主
- 移除管理员后自动撤销其管理员身份
- 移除后触发系统消息和当事人推送

## 35. 群成员管理-批量移除

### 页面概述
批量选择多个成员并统一移除，需二次确认。

### 组件清单

| 组件 | 说明 |
|------|------|
| 成员列表 | 带复选框 |
| 全选按钮 | 选择当前页全部 |
| 已选数量 | 显示已选人数 |
| 确认移除按钮 | 二次确认后移除 |

### 数据字段

| 字段 | 类型 | 说明 |
|------|------|------|
| 被移除成员ID列表 | 数组 | 要移除的成员集合 |

### 业务规则
- 仅管理员/群主可操作
- 移除需二次确认
- 每个被移除成员收到相同推送

## 36. 群成员管理-申请列表

### 页面概述
展示入群申请列表，支持单个/批量同意或拒绝。

### 组件清单

| 组件 | 说明 |
|------|------|
| 申请列表 | 展示申请人、申请理由、申请时间 |
| 同意按钮 | 同意单个申请 |
| 拒绝按钮 | 拒绝单个申请 |
| 全部同意 | 一键同意所有申请 |
| 全部拒绝 | 一键拒绝所有申请 |

### 数据字段

| 字段 | 类型 | 说明 | 校验规则 |
|------|------|------|---------|
| 申请人ID | 字符串 | 申请人唯一标识 | — |
| 申请理由 | 文本 | 入群理由 | ≤1000字，不支持图片 |
| 申请时间 | 时间戳 | 申请提交时间 | — |
| 处理状态 | 枚举 | 待处理/已同意/已拒绝 | — |

### 业务规则
- 申请理由不超过 1000 字，不支持图片
- 提交后 Toast 提示"已经成功提交申请"
- 支持单个同意/拒绝，支持一键全部同意/全部拒绝
- 有新申请时入口显示红点
- 群满员时即使同意也加入失败，需提示
