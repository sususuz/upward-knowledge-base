# 页面知识：赛事/活动/榜单/用户主页模块

> 模块：赛事、活动、榜单、用户主页
> 包含页面：赛事报名完成（3）、活动领队-行程入口（4）、活动成员-订单入口（5）、查看创作者主页-圈子入口（6）、查看用户主页-圈子入口（7）、榜单入口（43）
> 关联业务规则：`../business-rules/circle-types.md`

---

## 3. 赛事报名完成

### 页面概述
赛事报名成功后的页面，展示赛事信息，并自动加入赛事圈。

### 组件清单

| 组件 | 说明 |
|------|------|
| 报名成功提示 | 提示用户报名成功 |
| 赛事信息 | 赛事名称、时间、地点等 |
| 赛事圈入口 | 点击进入赛事圈 |
| 返回首页 | 返回赛事列表 |

### 业务规则
- 赛事上线后自动创建赛事圈
- 用户报名成功后自动加入赛事圈
- 赛事结束后 30 天自动归档

---

## 4. 活动领队-行程入口

### 页面概述
活动领队视角的行程入口，可查看活动圈、管理行程。

### 组件清单

| 组件 | 说明 |
|------|------|
| 活动圈入口 | 进入活动群聊 |
| 行程管理 | 管理活动行程 |
| 成员管理 | 查看活动成员 |

### 业务规则
- 活动有人报名成功后自动创建活动圈
- 圈主为活动领队
- 活动结束后 14 天自动关闭

---

## 5. 活动成员-订单入口

### 页面概述
活动成员视角的订单入口，展示订单信息，并进入活动圈。

### 组件清单

| 组件 | 说明 |
|------|------|
| 订单信息 | 活动订单详情 |
| 活动圈入口 | 进入活动群聊 |
| 联系领队 | 快速联系领队 |

### 业务规则
- 用户报名活动后自动加入活动圈
- 可在活动圈咨询领队行程问题

---

## 6. 查看创作者主页-圈子入口

### 页面概述
创作者个人主页展示粉丝圈入口，用户可关注并加入粉丝圈。

### 组件清单

| 组件 | 说明 |
|------|------|
| 创作者信息 | 头像、昵称、认证标识 |
| 粉丝圈入口 | 点击进入粉丝圈 |
| 关注按钮 | 关注创作者 |

### 业务规则
- 创作者认证成功后自动创建粉丝圈
- 关注创作者后自动加入粉丝圈
- 加入粉丝圈后自动关注创作者

---

## 7. 查看用户主页-圈子入口

### 页面概述
普通用户主页展示其主动创建的圈子入口。

### 组件清单

| 组件 | 说明 |
|------|------|
| 用户信息 | 头像、昵称 |
| 圈子列表 | 展示用户创建的圈子 |
| 加入按钮 | 加入/申请加入圈子 |

### 业务规则
- 普通用户主页显示他人主动创建的圈子
- 点击圈子可进入申请/加入流程

---

## 43. 榜单入口

### 页面概述
榜单相关圈子的入口，挑战者可进入榜单圈。

### 组件清单

| 组件 | 说明 |
|------|------|
| 榜单信息 | 榜单名称、时间、规则 |
| 榜单圈入口 | 进入榜单圈 |
| 挑战按钮 | 参与打榜 |

### 业务规则
- 榜单上线后自动创建榜单圈
- 挑战者自动加入榜单圈
- 榜单结束后 14 天自动关闭
