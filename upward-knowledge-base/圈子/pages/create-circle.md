## 14. 创建圈子

### 页面概述
用户创建新的用户群/达人圈，填写圈子基本信息、标签、加入设置、封面等。

### 截图区
- 默认态：表单初始状态
- 填写中：用户输入信息
- 校验失败：字段错误提示
- 提交成功：创建成功，跳转圈子主页/群聊

### 组件清单

| 组件名称 | 类型 | 说明 |
|---------|------|------|
| 圈子名称 | 输入框 | 圈子名称，≤20字符 |
| 圈子简介 | 多行输入 | 0/500字符 |
| 标签选择 | 标签组 | 游泳、健身、跑步、潜水、徒步、羽毛球、骑行、其他、爬山、越野 |
| 所在市/区 | 选择器 | 点击进入选择市区 |
| 加入设置 | 开关 | 加入圈子需要圈主确认 |
| 封面上传 | 上传组件 | 点击上传，压缩裁剪，未上传显示默认图 |
| 创建按钮 | 按钮 | 提交创建 |

### 用户操作路径

| 操作 | 结果 |
|------|------|
| 输入圈子名称 | 实时校验长度 |
| 输入圈子简介 | 显示字数统计 |
| 选择标签 | 可多选 |
| 点击所在市/区 | 进入选择市区页面 |
| 切换加入设置 | 控制是否需要审核 |
| 点击封面上传 | 选择图片并裁剪 |
| 点击创建 | 校验通过后创建圈子 |

### 数据字段

| 字段 | 类型 | 说明 | 校验规则 |
|------|------|------|---------|
| 圈子名称 | 文本 | 必填 | 非空，≤20字符 |
| 圈子简介 | 文本 | 选填 | ≤500字符 |
| 标签 | 数组 | 选填 | 多选 |
| 所在市 | 字符串 | 必填 | 已选择市 |
| 所在区 | 字符串 | 必填 | 已选择区 |
| 加入需审核 | 布尔 | 必填 | 默认false |
| 封面图片 | 图片 | 选填 | 未上传使用默认封面 |

### 业务规则
- 普通用户最多创建 10 个圈子
- 达人最多创建 50 个圈子
- 创建成功后创建者自动成为群主
- 加入用户群默认可能无需审核，由开关控制

## 15. 创建圈子-选择市区

### 页面概述
创建圈子时选择圈子所在的市和区。

### 组件清单

| 组件 | 说明 |
|------|------|
| 城市列表 | 按字母/热门排序 |
| 搜索框 | 搜索城市 |
| 区列表 | 选择市后展示该区列表 |
| 确认按钮 | 确认选择 |

### 数据字段

| 字段 | 类型 | 说明 |
|------|------|------|
| 市编码 | 字符串 | 城市唯一编码 |
| 市名称 | 字符串 | 城市名称 |
| 区编码 | 字符串 | 区唯一编码 |
| 区名称 | 字符串 | 区名称 |
