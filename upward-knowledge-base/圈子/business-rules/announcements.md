# 业务规则：群公告

> 规则编号：BR-AN-001
> 关联模块：群公告展示页、新建页面、内容页、权限体系
> 来源：墨刀原型解析 + 用户确认（标注 7、12、13、14）

---

## 1. 规则描述

群公告用于向群成员发布重要信息，如活动通知、安全规则、装备清单等。公告有数量上限、内容限制、附件限制、置顶规则，仅群主/管理员可管理。

---

## 2. 公告核心规则

| 规则项 | 内容 |
|--------|------|
| 创建权限 | 仅群主、管理员 |
| 查看权限 | 全员可见 |
| 最大数量 | 最多创建 5 个公告 |
| 置顶规则 | 最多 1 条公告置顶；新置顶替换旧置顶 |
| 新人必看 | 默认新人进群必看置顶/指定公告 |
| 编辑/删除 | 仅群主、管理员 |
| 内容长度 | 最多 1 万字符 |

---

## 3. 公告内容支持

| 类型 | 本期支持 | 说明 |
|------|---------|------|
| 文本 | ✅ | 最多 1 万字符 |
| 图片 | ✅ | jpg/png/webp，单张 ≤20MB，最多 10 张 |
| 文件 | ✅ | pdf/doc/docx/xls/xlsx/ppt/pptx/txt，单个 ≤100MB，最多 2 个 |
| 链接 | ✅ | 支持链接卡片 |
| 视频 | ✅ | mp4/mov，单个 ≤200MB，1 个/公告 |
| 轨迹 | ❌ | 二期规划 |
| 定位 | ❌ | 二期规划 |
| @ 人 | ❌ | 本期不支持 |

---

## 4. 公告附件限制

| 类型 | 允许格式 | 单公告限制 |
|------|---------|-----------|
| 图片 | jpg/png/webp | 单张 ≤20MB，最多 10 张 |
| 视频 | mp4/mov | 单个 ≤200MB，1 个/公告 |
| 文档 | pdf/doc/docx/xls/xlsx/ppt/pptx/txt | 单个 ≤100MB，最多 2 个 |
| 运动数据（二期） | gpx/tcx/fit | 单个 ≤20MB，最多 1 个 |
| **总量硬顶** | — | 单公告附件总大小 ≤300MB |

---

## 5. 公告状态与操作

### 5.1 展示页状态
- **空白状态**：显示"新建群公告"按钮
- **非空白状态**：展示公告列表，每个公告卡片显示标题、摘要、时间
- **置顶公告**：置顶区展示，新成员入群必看

### 5.2 公告卡片操作菜单
- **取消置顶**：取消当前置顶状态
- **设为新人进群必看**：指定公告为新人必看
- **删除公告**：删除该公告（二次确认）

### 5.3 新建/编辑公告
- 编辑器支持文本、图片、文件、链接插入
- 编辑器底部工具栏显示可插入内容类型
- 插入内容示例可能显示为"微信用户 177..."等占位，但实际本期不支持 @ 人或联系人

---

## 6. 状态流转

```
空白展示页
    │
    ▼
新建群公告
    │
    ▼
公告编辑中
    │
    ▼
发布公告 ──► 置顶公告（可选）
    │
    ▼
展示页列表
    │
    ▼
编辑 / 删除 / 取消置顶 / 设为新人必看
```

---

## 7. 例外情况

| 场景 | 处理规则 |
|------|----------|
| 公告数量达到 5 个 | 新建按钮置灰或提示"公告数量已达上限" |
| 附件总大小超过 300MB | 上传失败，提示"附件总大小超过限制" |
| 单张图片超过 20MB | 提示压缩或选择其他图片 |
| 群被解散 | 全部公告随群消息一起删除，不可恢复 |
| 群被关闭 | 公告仍可查看，不可新建/编辑 |

## 9. 测试用例重点

1. 公告数量上限 5 个，超出后正确拦截。
2. 附件类型、大小、数量边界（图片 20MB×10、视频 200MB×1、文档 100MB×2、总量 300MB）。
3. 置顶公告替换规则，新人入群必看推送。
4. 群主/管理员可创建/编辑/删除，普通用户不可操作。
5. 群关闭后公告只读，群解散后公告删除。
6. 公告内容 1 万字符边界，空内容/纯空格提交校验。
