## 二、圈子类型与自动创建逻辑

| 圈子类型 | 创建触发条件 | 圈主身份 | 默认简介 | 自动加入规则 |
|----------|-------------|----------|----------|-------------|
| **赛事圈** | 赛事上线后自动创建 | 赛事主办方/官方账号 | "赛事选手群，重要赛事通知实时更新，供参赛选手交流互助。" | 用户报名成功后自动加入 |
| **活动圈** | 活动有人报名成功后自动创建 | 发起活动的领队 | "同行伙伴交流区，行程问题可咨询领队。" | 用户报名成功后自动加入 |
| **粉丝圈** | 创作者认证成功后自动创建 | 创作者本人 | "欢迎来到 XX 的粉丝交流群！" | 用户关注创作者后自动加入 |
| **用户群** | 普通用户/达人主动创建 | 创建者本人 | 自定义 | 需申请/邀请加入 |
| **榜单圈** | 榜单上线后自动创建 | 榜单创建者 | — | 挑战者自动加入 |

## 五、群聊功能详细规则

### 5.1 消息类型与限制

| 消息类型 | 长度/大小限制 |
|----------|--------------|
| 文本 | 单条 ≤2000字 |
| 图片 | 单图 ≤10MB，最多9张 |
| 视频 | ≤500MB，时长 ≤5分钟 |
| 语音 | ≤60秒 |
| 文件 | ≤200MB |
| 链接 | 无限制 |
| 位置 | — |

### 5.2 消息操作菜单权限矩阵

| 菜单项 | 自己的消息 | 他人的消息 | 群主/管理员看他人消息 |
|--------|-----------|-----------|---------------------|
| 引用 | ✅ | ✅ | ✅ |
| 复制 | ✅（文本类） | ✅（文本类） | ✅ |
| 多选 | ✅ | ✅ | ✅ |
| 撤回 | ✅（2分钟内） | ❌ | ✅（无时限） |
| 转发 | ✅ | ✅ | ✅ |
| 举报 | ❌ | ✅ | ✅ |
| 置顶 | ❌ | ❌ | ✅ |
| 删除（他人非本地） | ❌ | ❌ | ✅ |

### 5.3 各操作功能详解

#### 引用
- **触发**：长按消息 → 主菜单"引用"
- **适用**：全部类型（文本/图片/视频/语音/文件/链接/活动卡片/位置/动态卡片）
- **缩略规则**：文本>50字截断显示前50字+…；非文本显示类型标识如`[图片]`、`[语音 0:23]`、`[活动卡片：周六晨跑]`
- **嵌套规则**：不支持嵌套，只引用最外层
- **跳转**：点击引用块 → 高亮跳转原消息（黄色闪烁1.5s）
- **原消息撤回**：显示"原消息已被撤回"，可点击但不跳转
- **必须带文本**：不允许空引用

#### 撤回
- 普通成员：发送后 **2分钟内** 可撤回
- 群主/管理员：**无时限** 限制
- 撤回提示：全员可见"XXX 撤回了一条消息"

#### 复制
- 适用：文本、链接（复制URL）、位置（复制地名+详细地址）
- 不适用：纯图片/视频/文件/语音 → 替换为"保存"
- **风控提示**：检测到手机号/银行卡号/身份证 → Toast提示"内容含个人信息，请谨慎使用"（不阻断）

#### 多选
- 可批量操作：逐条转发、合并转发、删除、举报
- 不可批量操作：引用、撤回、置顶（单条语义操作）
- **选择上限**：99条/次
- 已撤回消息不可选（勾选框置灰），系统消息不可选

#### 转发
- 适用：文本/图片/视频/语音/文件/链接卡片/位置
- 不可转发：被撤回的、系统消息
- 模式：逐条转发（每条独立显示在目标会话）
- **目标数量限制**：单次最多选 **9个** 目标群/人
- 附言：限200字

#### 置顶
- 权限：仅群主/管理员
- **置顶上限**：默认 **5条** 置顶消息 + 1条置顶公告
- 达上限后：二次确认弹窗 → 替换最早置顶的消息
- 不可置顶：已撤回的、被举报隐藏的、系统消息、自己已置顶过的
- **新成员入群**：5秒内系统以单聊推送"群当前置顶消息"全文

#### 举报
- 不可举报自己的消息和系统消息
- **举报理由**（必选1项）：垃圾广告、色情低俗、政治敏感、人身攻击/辱骂、谣言/虚假信息、涉嫌诈骗、售卖违禁品、侵犯隐私、与赛事/活动无关刷屏、其他（自由文本限200字）
- 附加信息：补充说明（选填500字）+ 截图证据（选填最多3张）
- **三级处理流程**：
  - L1 群主审核：24h内处理
  - L2 平台升级：群主未处理自动升级
  - L3 平台反馈：24h内反馈结果
- **特殊升级**：选择"涉嫌诈骗"/"政治敏感"/"色情低俗" → 跳过L1直接进L2
- **反恶意举报**：同一用户1h内最多举报10条；同一消息只能举报1次；连续3次无效 → 当日熔断+信用分扣减

#### 删除
- 权限：群主/管理员
- 作用对象：他人的消息
- **不可恢复**
- 二次确认弹窗

### 5.4 聊天界面时间戳规则

| 时间间隔 | 显示规则 |
|---------|---------|
| 进入聊天后第一条可见消息 | 强制显示时间戳 |
| 跨日切换（跨过00:00） | 强制显示日期分隔条 |
| 与上一条间隔 ≥5分钟 | 显示时间戳 |
| 其他情况 | 不显示 |

### 5.5 时间显示格式

| 距今时长 | 显示格式 | 示例 |
|---------|---------|------|
| <1分钟 | 刚刚 | 刚刚 |
| 当天 | HH:mm | 14:30 |
| 昨天 | 昨天 HH:mm | 昨天 21:45 |
| 前天 | 前天 HH:mm | 前天 10:20 |
| 本周内（3-7天前） | 星期X HH:mm | 星期二 14:30 |
| 本年内（8天-当年内） | M月D日 HH:mm | 3月15日 14:30 |
| 跨年 | YYYY年M月D日 HH:mm | 2024年12月20日 14:30 |

**列表中的时间显示**：

| 距离今天 | 显示格式 |
|---------|---------|
| 当天 | 15:30 / 下午 3:30 |
| 昨天 | 昨天 |
| 本周内（2-6天前） | 星期三 / 周三 |
| 超过一周 | 10月1日 或 2024/10/1（跨年） |

### 5.6 特殊场景处理

| 场景 | 处理方式 |
|------|---------|
| 群被关闭 | 卡片标灰，标注"群已关闭"，仅保留消息历史可看 |
| 群被解散 | 卡片标灰，标注"群已解散"，3天后自动消失 |
| 用户被踢出 | 卡片标灰+"你已被移出群"，3天后自动消失 |

## 六、权限体系

| 角色 | 权限 |
|------|------|
| **普通用户** | 查看群成员、修改群昵称、退出群聊 |
| **管理员** | 普通用户权限 + 添加/删除成员、申请审批、群公告、加入审核、关闭/重启群聊 |
| **群主** | 管理员权限 + 转让群主、指定管理员、解散群聊（不可退群，需先转让） |

**管理员设置规则**：
- 最多设置 **3个** 管理员
- 不显示群主本人
- 现存管理员默认被选择
- 超出上限提示"管理员人数超过上限"

**转让群主**：
- 只能选择一个新群主
- 群主异常移交（封号）→ 自动移交给最早的管理员

## 七、群公告功能

### 7.1 公告规则
- 最多创建 **5个** 公告
- 最多 **1条** 公告置顶，新置顶替换旧置顶
- 默认新人进群必看
- 只有群主/管理员可创建/编辑/删除
- 公告支持发送图片、文件、链接（第一期暂不支持轨迹、定位）

### 7.2 公告内容限制
- 最多输入 **1万字符**
- 支持插入图片、文件

### 7.3 公告附件限制

| 类型 | 允许格式 | 单公告限制 |
|------|---------|-----------|
| 图片 | jpg/png/webp | 单张 ≤20MB，最多10张 |
| 视频 | mp4/mov | 单个 ≤200MB，1个/公告 |
| 文档 | pdf/doc/docx/xls/xlsx/ppt/pptx/txt | 单个 ≤100MB，最多2个 |
| 运动数据（二期） | gpx/tcx/fit | 单个 ≤20MB，最多1个 |
| **总量硬顶** | — | 单公告附件总大小 ≤300MB |

## 八、成员管理

### 8.1 邀请成员
- 可选人数上限为群上限
- 已是群成员不可被选择

### 8.2 移除成员
- 只有管理员和圈主可操作
- 支持批量移除
- 移除需二次确认

### 8.3 申请加入审批
- 申请理由不超过 **1000字**，不支持图片
- 申请后Toast提示"已经成功提交申请"
- 支持单个同意/拒绝，支持一键全部同意/全部拒绝
- 赛事和活动群默认需审核；通过报名加入的不需要审核
- 有新申请时显示红点

## 九、圈子发现与推荐

### 9.1 推荐排序逻辑
1. **优先展示同城圈子**
2. **按用户标签匹配度推荐**
3. **按圈子热度和创建时间推荐**

### 9.2 搜索功能
- 对圈子名称和圈主名称模糊搜索
- 搜索页可切换Tab页签，切换后保持关键字
- 输入文字后**实时搜索**
- **高亮**搜索文字

### 9.3 圈子列表排序
- 按最新消息时间**倒序**排序
- 未读消息显示红点，点击进入后消失
- 群聊条数实时显示，最多显示 **99+**

### 9.4 创建圈子表单
- **圈子名称**：不超过20个文字字符
- **圈子简介**：0/500字符
- **标签**：游泳、健身、跑步、潜水、徒步、羽毛球、骑行、其他、爬山、越野（可多选，根据用户标签设置显示）
- **所在市/区**：选择
- **加入设置**：加入圈子需要圈主确认（开关）
- **封面**：点击上传，对图片压缩和裁剪，未上传显示默认图片
